Story time! How this was made.

This was originally a project for work where the client needed a dynamic search form and I thought the solution I came up with was pretty slick.

A brief overview of how it works is:

  1. I add all the options into an array
  2. I start a loop
  3. I start a loop
  4. I start a loop....just kidding
  5. Just one loop
  6. I splice out the first element of the array
  7. I do that until the index is greater than the number of max inputs per row
  8. The rest of the array are the "advanced options"

I thought that was pretty clever so I decided to build this little page to show if off!
